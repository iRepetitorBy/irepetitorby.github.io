Java для начинающих

Java используют около 10 миллионов разработчиков во всем мире.
Java делит с JavaScript звание самого популярного языка программирования.
Следует подчеркнуть, что Java - это не столько язык, сколько крупнейшая платформа, позволяющая решать практически любые задачи.
Количество материалов по Java огромно. И, в то же время, отсутствует единая "библия", изучив которую можно стать хотя бы junior Java developer, и которая, в то же время была бы доступной и понятной для изучения.
Поэтому для начинающих я рекомендую начать с изучения следующих материалов:


Java Programming for Kids, Parents and Grandparents
by YAKOV FAIN

Данная книга доступна на английском и русском языках. Причем русская версия даже лучше английской.
Я рекомендую начать именно со "старой" версии книги, поскольку в последней версии книги (The New Edition [2015]) вместо используемого в реальных legacy приложениях Swing примеры даны на мертворожденной JavaFX.

Следует отметить, что:
  • Java сравнительно редко используется для написания UI - это язык серверов.
  • Server-side Java и Java на Android это две совершенно разные платформы.

Далее параллельно следует использовать сразу две книги:


Head First Java
by Bert Bates, Kathy Sierra

Следует отметить, что раньше эта книга рекомендовалась как обязательная к изучению перед прохождением базового курса по Java в Образовательном центре Парка высоких технологий.


Java. Методы программирования.
И.Н. Блинов, В.С. Романчик
Минск: издательство «Четыре четверти», 2013. — 896 с.
